# FastAPI Basic Project

This is a basic FastAPI project.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the server:
   ```bash
   uvicorn main:app --reload
   ```
   or 
   ```bash
   python main.py
   ```

Visit http://127.0.0.1:8000 in your browser to see the API.
